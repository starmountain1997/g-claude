# 场景细纲构建器

> novel-writter 第二阶段：从分章大纲到场景级细纲。
> 上一阶段 → `outline`（创作大纲），下一阶段 → `writing`（正文写作）。

将分章大纲中的每一章拆解为 **3-5 个场景**，逐场规划：
剧情起伏线、人物情感线、人物立场变化和整体形势演进。

大纲告诉你"发生了什么"，细纲告诉你"怎么发生的"——前者是卫星地图（看清全貌），
后者是街区地图（看清每一步怎么走）。

---

## 输入与前置条件

细纲的输入是 `outline` 子技能产出的**已确认的分章大纲**，包含：
- 世界观设定、人物设定、人物关系网络
- 主线与支线故事线
- 分章大纲（每章概要、POV、核心冲突、情节/人物推进、伏笔）

**如果输入不完整**，先回到 `outline` 补充缺失部分，不要在没有完整大纲的情况下强行拆场景。

---

## 工作流程

### 阶段零：消化大纲，建立全局坐标系

拆场景之前，先通读大纲并标记：

1. **每章的核心戏剧功能**——这章在故事中承担什么角色？（铺垫 / 转折 / 高潮 / 收束 / 呼吸调节）
2. **每个人物的情感起点和终点**——从大纲中提取每条成长弧线的两端
3. **主线的关键转折点位置**——激励事件、中点反转、至暗时刻、高潮、落点分别落在哪一章
4. **所有伏笔清单**——每条伏笔的埋设章和预计回收章

然后画一条**张力预期草图**（不必精确，目的是建立"位置感"）：

```
章序:   一     二     三     四     五     六     七     八
       ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤
张力
高                           ╱╲              ╱╲
中                 ╱╲      ╱  ╲    ╱╲     ╱  ╲
低    ╱╲        ╱╲  ╲  ╱╲    ╲  ╱  ╲   ╱    ╲
      激励      转折₁    中点    至暗   转机   高潮   落点
```

这个草图在拆场景过程中会不断修正，但保证你拆每一场戏时都有"我在哪里"的意识。

### 阶段一：逐章拆解场景

对每一章，拆解为 3-5 个场景。每个场景填写以下模板。

#### 场景模板

```
### 第X章 · 场景N：[场景标题]
（标题暗示本场的核心戏剧动作，如"谈判破裂"而非"办公室对话"）

- **场景功能**：
  本场在故事中的作用（铺垫伏笔 / 推进主线 / 揭示人物 / 制造转折 /
  收束支线 / 情绪调节）。如果删掉这场戏，故事线会在哪里断裂？

- **地点/时间**：[具体场景 + 时间节点（时刻/日期/相对时间）]

- **出场人物**：[谁在场？谁不在但通过对话/回忆/信息被提及？]

- **POV**：[通过谁的感知系统讲述？为什么这个视角在此刻最有张力？]

- **入场状态**：
  - **整体形势**：[本章/本场开始时的宏观局面——
    各方势力分别占据什么位置？谁掌握主动权？
    关键的信息不对称是什么？是否有时间压力/倒计时？]
  - **人物立场**：
    · [人物A]：[此刻的短期目标] / [对当前局面的判断和态度] /
      [信任谁 ↔ 戒备谁]
    · [人物B]：[此刻的短期目标] / [对当前局面的判断和态度] /
      [信任谁 ↔ 戒备谁]
    （列出所有出场及本场被涉及的关键人物）

- **冲突引擎**：
  - **冲突类型**：外部冲突 / 内部冲突 / 关系冲突（可多选）
  - **冲突核心**：谁想要什么？谁或什么在阻挡？双方各自的筹码是什么？
  - **冲突强度**：1-10
    （1-3=暗流涌动，对话中有潜台词但未摊牌；
     4-6=正面交锋，言语或行动上的对抗；
     7-9=剧烈冲突，有人受伤或局面不可逆；
     10=生死关头或终极抉择）

- **情感节拍**（本场的情感推进轨迹）：
  · [人物A]：入场情绪 → 触发事件 → 情绪转变（为何变/怎么变）→ 出场情绪
  · [人物B]：入场情绪 → 触发事件 → 情绪转变（为何变/怎么变）→ 出场情绪
  （情绪用具体描述而非抽象标签——
   不说"愤怒"，说"被最信任的人欺骗后的那种冷静的恨意"）

- **核心节拍序列**（3-5 个关键动作/对话/决定）：
  1. **开场钩子**：本场的前 30 秒——什么立刻抓住注意力？（一句异常的话？
     一个反常的动作？一个意外的发现？）
  2. **冲突展开**：对抗/揭示/谈判如何步步推进？
  3. **转折点**：本场中什么改变了游戏规则？（一次选择、一个信息、
     一个事件）
  4. **退场钩子**：本场最后留在读者心里的问题或画面是什么？

- **信息释放**：
  - **读者新知道了什么**：
  - **什么被进一步悬置**（谜加深或新谜出现）：
  - **伏笔操作**（埋设/推进/回收）：

- **退场状态**：
  - **整体形势变化**：[本场结束后局面发生了什么可感知的改变]
  - **人物立场变化**：[谁的短期目标变了？谁的态度/信任关系发生了偏移？
    谁没变（同样重要）？]
  - **退场悬念**：[本场结束时，读者最想得到答案的问题是什么]

- **叙事提示**（给 `writing` 阶段的建议，写作时参考即可）：
  - **节奏**：快 / 慢 / 张弛交替
  - **叙述距离**：远景（概括叙述）/ 中景（场景呈现）/ 近景（内心独白或特写）
  - **必须着力之处**：这场戏中不能一笔带过的细节/动作/对话
  - **弹性空间**：哪些描写、过渡、次要对话可由写作时自由发挥
```

### 阶段二：绘制全局跟踪线

所有章节的所有场景拆解完成后，从四个维度绘制全局跟踪图。
**这是细纲质量的试金石——单场戏写得好不代表整体节奏对。**

#### 2.1 剧情起伏线（张力曲线）

将每个场景的冲突强度标在一条时间轴上：

```
场景: 1-1 1-2 1-3 2-1 2-2 2-3 3-1 3-2 3-3 4-1 ...
强度:  3   5   2   4   7   6   5   8   6   9  ...
       ↑       ↑           ↑           ↑
     激励    首次受挫     中点反转    至暗时刻
```

**审视清单：**
- [ ] 整体是否呈上升趋势（回落是为了更高的爬升，而非真的泄气）？
- [ ] 连续 3 个以上场景强度持平？（→ 需要变化）
- [ ] 高潮场的强度是否确实是全局最高点？
- [ ] "呼吸场景"（强度 2-3）的位置是否合理——是否给了读者消化冲击的间隙？
- [ ] 强度 8-10 的场景是否有足够的情节重量支撑（不是标得高就真的高）？
- [ ] 是否有"假高潮"——一个强度 7 的场景后紧接着更高的 9，导致前者被冲淡？

#### 2.2 人物情感线

为每个核心人物画一条情感弧线，逐场标注：

```
[人物A] 情感弧线
       1-1    1-2    1-3    2-1    2-2    2-3    3-1    3-2   ...
情绪: 平静 → 焦虑 → 愤怒 → 决绝 → 动摇 → 崩溃 → 麻木 → 重生  ...
       ───────────────────────────────────────────────────────
       弧线方向：从 "对世界抱有天真信任" → "学会带着伤疤继续前行"

触发:        1-2被  1-3被  2-1发现  2-2计  2-3遭  3-1失  3-2在
             欺骗   背叛   别无选择  划失败  重创   去一切  废墟中找到
                                                    新的理由
```

**审视清单：**
- [ ] 每次情绪转变是否有具体事件触发（不是凭空变脸）？
- [ ] 情感是否有层次和多样性（不是从头到尾一个调子）？
- [ ] 是否存在情感跳跃——前一场到后一场情绪突变但缺乏触发事件？
- [ ] 配角的情感线是否也在推进（不只有主角在感受，配角也有自己的情感旅程）？
- [ ] 情感高潮是否与剧情高潮有交错——最激烈的外部事件 × 最深刻的内部触动，
  二者在时间上是否形成了有意义的呼应（可能同步，也可能有意错开）？
- [ ] 人物之间的情感线是否有交叉影响（A 的情绪变化如何触发 B 的情绪反应）？

#### 2.3 人物立场演进表

逐场记录每个关键人物的立场变化：

| 场景 | 人物A立场 | 人物B立场 | 人物C立场 | 权力格局 |
|------|----------|----------|----------|---------|
| 1-1 | 天真信任B，目标：找出真相 | 表面协助A，暗中引导方向 | 观望，不站队 | A≈B>C |
| 1-2 | 开始怀疑B的动机 | 加紧控制信息流 | 向A透露部分真相 | B>A>C |
| 1-3 | 发现B的欺骗，立场反转 | 摊牌，转为公开对抗 | 表态支持A | A+C≈B |
| ... | ... | ... | ... | ... |

**立场维度包括：**
- **目标**：短期要达成什么
- **同盟/敌对关系**：此刻与谁联手，与谁对立
- **信息立场**：知道什么 vs 被蒙蔽什么
- **道德立场**：此刻的自我辩护逻辑——ta 认为自己"做得对"的理由是什么

**审视清单：**
- [ ] 立场变化是否有因果链（因为 X 事件/Y 信息，所以 Z 改变了立场）？
- [ ] 是否存在"僵化立场人物"（从头到尾立场不变，且没有给出内在坚持的理由）？
- [ ] 权力格局是否在动态演变（如果永远一边倒，冲突从何而来）？
- [ ] 人物的信息立场是否合理（ta 获取信息的渠道是否与 ta 的处境一致）？
- [ ] 同盟关系的形成和破裂是否有足够的铺垫？

#### 2.4 整体形势演进图

逐场或逐章记录宏观局面的变化：

```
场景:  1-1     1-2     1-3     2-1     2-2     2-3     3-1 ...
─────────────────────────────────────────────────────────────
形势: 表面    第一道   多方    阵营    A方遭   形势     B方内
      和平    裂痕    势力    开始    受第一   进入    部出现
      各怀    出现    开始    成形    次打击   僵局    裂痕
      心思          浮出
                    水面
─────────────────────────────────────────────────────────────
信息: 只有B   A开始    A发现   C公布   A掌握   B发现   所有人都
差:   掌握全   怀疑    部分    关键    关键    C已经   知道——
      貌              真相    证据    证据   倒向A   但知道的是
                                                  不同版本
```

**审视清单：**
- [ ] 形势的每次变化是否"可感知"（读者能明确感受到局面不同了）？
- [ ] 信息差的演变是否构成独立的悬念线？
- [ ] 是否存在"形势静止"的区间（长时间没有变化→需要打破）？
- [ ] 高潮前是否形成了"最坏局面"——主角在所有维度上都处于劣势？

### 阶段三：自我审视修正

完成所有场景拆解和跟踪图后，至少进行 **2 轮**自我审视。

#### 第一轮：场景级质量

逐场检查：

- [ ] **独立性**：每场戏是否有独立的冲突引擎？删除这场戏，故事线会在哪里断裂？如果找不到断裂点，这场戏可能不需要。
- [ ] **反信息倾泻**：是否存在大段说明、缺乏冲突和动作的场景？如有，将说明性内容融入冲突场景中释放。
- [ ] **过渡承接**：前一场的退场钩子是否在下一场被承接（不一定是直接继续，但要有因果/情感上的关联）？
- [ ] **合并可能**：是否有两场功能重叠、地点相同、人物相同的场景可以合并？
- [ ] **缺失检查**：是否有情感转折需要一场独立的戏来支撑，但目前被跳过了（如"他想通了"只标注但没有任何一场戏展示他如何想通）？
- [ ] **POV 最优**：当前场景用谁的视角讲最有张力？有没有换个 POV 会让信息释放更精彩的场景？

#### 第二轮：全局一致性

- [ ] 起伏线是否合理——整体爬升，回落有度，没有"平淡集中区"？
- [ ] 每条人物情感线是否完整——有起点、有触发、有转折、有落点？
- [ ] 伏笔的埋设和回收是否全部在场景中安放了明确位置？
- [ ] 支线与主线的场景比例是否合理——支线不喧宾夺主，也不彻底消失？
- [ ] 信息释放节奏——读者是否始终处于"知道刚好够来保持好奇但不会完全迷失"的状态？

#### 第三轮（可选，长篇或复杂故事）：读者体验模拟

- [ ] 如果连续阅读场景序列，是否产生"再读一场就睡结果读了十场"的效果？
- [ ] 是否有读者会本能想跳过的场景？（→ 说明场景功能太弱或节奏太拖）
- [ ] 情感体验是否丰富——不只有紧张，还有温暖、幽默、悲伤、释然？
- [ ] 埋了 3 个以上未回收伏笔后，是否到了该回收一条的节点？

### 阶段四：提交用户审阅

提交格式：

1. **先给总览**——起伏线图 + 一句话总结细纲的核心设计思路
2. **逐章展示**——每章先列场景清单，再逐一展开
3. **标注关键设计选择**——如"此处让情感崩溃发生在外部事件解决之前，目的是让读者先共情再释然"
4. **提供 1-2 个备选方案**——如"转折点可以放在 3-2 也可以延迟到 3-4，前者更紧凑但牺牲铺垫。你觉得？"
5. **开放反馈**

## 核心原则

### 1. 每场戏都是一个微型故事
有自己的"起承转合"——开场钩子（吸引）→ 冲突升级（推进）→ 转折（改变方向）→ 退场钩子（悬念）。
不存在"纯粹过渡"的场景——过渡在场景内部完成。

### 2. 情感必须有来处和去处
不说"主角突然愤怒"→ 说"1-2 中的欺骗在 2-1 被主角证实，累积的怀疑在此刻爆发为愤怒，
而这种愤怒将在 2-3 推动他做出不可逆的选择"。
每滴眼泪、每次沉默、每次爆发，都应该在之前的场景中找到触发源，
在之后的场景中看到后果。

### 3. 立场是动态的，不是道德标签
立场是人物对"当下局面中我站在哪里、我要什么"的判断，不是好人/坏人的道德评分。
好人可以判断失误站错队，对手也可以在特定情境下伸出援手。
每次立场变化都必须有**可理解的原因**——不是因为大纲说"这里该变了"，而是因为发生了一件事、
获得了一个信息、或失去了一份信任。

### 4. 起伏线要有呼吸感
持续高强度 = 没有强度。好的起伏线像深呼吸——
紧张后放松，放松中埋下新的不安，让读者在安全感消失之前就感到下一波紧张的来临。
强度 2-3 的"呼吸场景"不是废戏，它们是读者消化冲击、人物重建内心、新矛盾暗中萌芽的空间。

### 5. 形势是动态的背景，不是静态的幕布
每一场戏结束后，整体形势都应有可感知的变化——
某个秘密不再安全了，某段联盟出现了裂痕，某个倒计时又近了一步，某方的筹码增加了或贬值了。
如果连续两场结束后形势完全不变，要么合并为一场，要么其中一场缺乏戏剧必要性。

### 6. 视角选择本身就是叙事
同一场戏用不同人物的视角讲述，效果天差地别。
选择 POV 时，问自己：这场戏中，谁的信息最不完整因而最有发现感？
谁的情感反应最强烈？谁处在"看到但无法阻止"的折磨位置？
谁距离真相最远/最近因而读者能通过 ta 的眼睛产生最大悬念或最大满足？
